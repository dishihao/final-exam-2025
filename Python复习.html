简答题 11. 解释Python中的break和continue语句的作用，并给出一个示例。

答：
- break：立即终止当前所在循环，跳出循环体。
- continue：跳过本次循环剩余代码，直接进入下一次迭代。
示例：
for i in range(5):
if i == 2: continue # 跳过2
if i == 4: break # 到4结束
print(i) # 输出 0, 1, 3

简答题 12. for循环和while循环有什么区别？

答：
- for循环：主要用于遍历已知序列（如列表、字符串）或固定范围（range）。
- while循环：主要用于在满足特定条件时重复执行，直到条件不再满足，通常用于循环次数不确定的情况。

简答题 变量作用域中局部变量和全局变量的区别?

- 局部变量：在函数内部定义，只在函数内有效，函数结束即销毁。
- 全局变量：在模块层级定义，整个程序生命周期内有效。函数内修改全局变量需用 global。
